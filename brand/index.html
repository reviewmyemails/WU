Color Usage Guidelines

✓ DO

  • Use Dark (#134e4a) for headers and important text
  • Use Accent (#0d9488) for CTAs and links
  • Use Light (#f0fdfa) for section backgrounds
  • Alternate dark/light sections for rhythm
  • Use status colors only for their purpose
  • Keep Gold (#fbbf24) for special highlights only
  • Use white text on dark backgrounds
  • Maintain high contrast for readability

✗ DON'T

  • Don't use red/danger for non-error elements
  • Don't overuse the accent color
  • Don't use gold as a primary color
  • Don't put dark text on dark backgrounds
  • Don't mix too many colors in one section
  • Don't use status colors for decoration
  • Don't create low-contrast text
  • Don't stray from the defined palette
