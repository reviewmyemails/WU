Section Container Component

Wrapper for page sections. Controls background color, vertical padding, and content width. Scroll down to see all variants.

.section--white .section--pad-lg

White Background, Large Padding

Default for most content sections. Clean and readable.

.section--light .section--pad-lg

Light Background, Large Padding

Use to alternate with white sections. Creates visual rhythm.

.section--dark .section--pad-lg

Dark Background, Large Padding

Use sparingly for emphasis. CTA blocks, footer-adjacent sections.

.section--white .section--pad-xl

Extra Large Padding

Reserved for hero sections. Maximum breathing room.

.section__inner--narrow

Narrow Content Width

Max-width 720px. Use for focused reading like long-form text, terms pages, or centered content blocks.

.section--pad-md

Medium Padding

Tighter spacing for smaller sections or when content is dense.

Usage

<section class="section section--white section--pad-lg">
  <div class="section__inner">
    <!-- Your content here -->
  </div>
</section>

Classes

Class Purpose
.section Base class (required)
.section--white White background (#fafcfc)
.section--light Light teal background (#ccfbf1)
.section--dark Dark teal background (#134e4a), white text
.section--pad-md Medium padding (48px desktop)
.section--pad-lg Large padding (80px desktop)
.section--pad-xl Extra large padding (120px desktop)
.section__inner Content container (1200px max)
.section__inner--narrow Narrow container (720px max)
.section__inner--full Full width, no padding

Page Rhythm Pattern

Alternate backgrounds to create visual flow:

Hero        → white, pad-xl
Problem     → light, pad-lg
Solution    → white, pad-lg
Features    → light, pad-lg
Testimonial → white, pad-lg
CTA         → dark, pad-lg
Footer      → dark

Responsive Behavior

  • Desktop: Full padding, 80px horizontal margins
  • Tablet: Reduced padding, 48px horizontal margins
  • Mobile: ~30% less vertical padding, 24px horizontal margins

Accessibility

  • Use semantic <section> element
  • Add id for anchor links
  • Dark sections maintain WCAG AA contrast
